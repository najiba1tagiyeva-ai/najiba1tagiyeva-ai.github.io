CC5. Accessing data: Scraper and API.

Chart 1

This visualization uses live data from the World Bank Open Data API.The API allows users to retrieve country-level indicators by specifying a country code, an indicator code, and a format parameter.

Base URL:
https://api.worldbank.org/v2/

Indicators used:
• Internet Usage Indicator: IT.NET.USER.ZS (Individuals using the internet, % of population)
• Mobile Subscriptions Indicator: IT.CEL.SETS.P2 (Mobile cellular subscriptions, per 100 people)

Final API URL Used (shortened):
https://api.worldbank.org/.../IT.NET.USER.ZS?date=2023

Show full API URL 
https://api.worldbank.org/v2/country/AGO;BDI;BEN;BFA;BWA;CAF;CIV;CMR;COD;COG;COM;CPV;ERI;ETH;GAB;GHA;GIN;GMB;GNB;GNQ;KEN;LBR;LSO;MDG;MLI;MOZ;MRT;MWI;NAM;NER;NGA;RWA;SDN;SEN;SLE;SOM;SSD;STP;SWZ;TCD;TGO;TZA;UGA;ZAF;ZMB;ZWE/indicator/IT.NET.USER.ZS?date=2023&format=json

The charts above retrieve data directly from the API through Vega-Lite lookups, merging country codes with indicator values in real time.

Chart 2

This visualization is built using scraped data from Wikipedia's UN Population Statistics. The data was collected using Python (pandas and requests), cleaned, and transformed into tidy (long) format.

Data Source:
Wikipedia - List of countries by population (United Nations)

Scraping Method:
• Python libraries: pandas, requests
• HTML table parsing with pd.read_html()
• Data cleaning and normalization

Data Format:
Tidy (long) format with columns: country, variable, value

View the data scraping notebook
